# 588710 缠论实时看板

把 **588710（科创半导体设备 ETF，华泰柏瑞）** 的缠论结构做成可本地运行的实时看板：实时报价跳动 + 五级（日/30分/15分/5分/1分）缠论笔/中枢/买卖点 + 多级别联立的「下一步操作建议」。

纯 Python 标准库，**零第三方依赖**，clone 后一行命令即可跑。

## 功能

- 🔴 **实时报价**：后端每 3 秒从新浪财经抓取 588710 现价 / 涨跌幅 / 开高低 / 时间，前端 ticker 实时刷新。
- 📊 **五级缠论结构**：日线 / 30分 / 15分 / 5分 / 1分 五档联立，笔、线段、中枢、买卖点全标注（数据来自 Vespa314/chan.py 预计算的缓存）。
- 💡 **下一步操作建议**：后端用多级别联立降维成白话——大级别定方向（偏多/偏空/震荡）与现价相对中枢的位置，小级别抓最近活跃买卖点信号，结合实时价给出动作参考，并附风险提示。
- 🖱️ **交互**：级别 Tab 切换、图层开关（笔 / 线段 / 中枢 / 买卖点 / 成交量 / MACD）、滚轮缩放、拖拽平移、悬浮看 K 线明细。

## 工作原理

```
┌─────────────────┐   每3秒    ┌──────────────────┐
│  新浪财经行情    │ ───────▶  │  realtime_server │
└─────────────────┘           │  (Python stdlib) │
                               │                  │
chan.py 预计算 ──▶ chanpy_data │  /api/snapshot   │ ──▶ 前端 Canvas 渲染
_5level.json      _5level.json│  ?full=1 / 轻量  │
                               └──────────────────┘
```

- **K 线结构**（`chanpy_data_5level.json`）：由 Vespa314/chan.py 离线计算好缠论标注后导出，本仓库直接内置这份数据。数据有更新时，替换该 JSON 即可，服务会在下次请求时按文件修改时间自动重载，无需重启。
- **实时价**：盘中走新浪接口，收盘后为静态快照。
- **操作建议**：纯前端展示后端算好的结果，缠论是技术参考，**不构成投资建议**。

## 接口

| 路径 | 说明 |
|------|------|
| `/` | 实时看板页面 |
| `/api/snapshot?full=1` | 全量：五级 bars + 缠论标注 + 实时价 + 操作建议（首屏加载用） |
| `/api/snapshot` | 轻量：实时价 + 操作建议 + live_price（前端轮询用） |

## 数据格式与导入

看板消费的底层数据是固定的「五级 bars + 缠论标注」JSON 结构（详见本仓库 `chanpy_data_5level.json`）。
更新数据时，用一份同结构的 JSON 替换 `chanpy_data_5level.json` 即可，服务按文件修改时间自动重载，**无需重启**，浏览器刷新即见。

> 核心约束：每条笔 / 线段 / 中枢 / 买卖点的时间字段必须**精确等于** `bars` 中某根的 `time`
> （如日线 `2026-07-07`、分钟线 `2026-07-07 15:00`），否则前端无法定位。

完整的「契约文档 + 校验器 + 导入器 + 示例」工具链在本地维护（未随公开仓库发布）。如需接入自有模型产出数据，可联系作者获取。
